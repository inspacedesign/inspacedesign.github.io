# 首頁照片資料夾

這裡放的是首頁大圖瀏覽區的照片，跟 case-XX 各自獨立。

## 命名規則

依序命名為 `01.jpg`、`02.jpg`、`03.jpg`……，必須連續不能跳號。

## 規格建議

- 副檔名一律 `.jpg`（不要 `.jpeg` / `.JPG`）
- 橫向比例（接近 3:2 或 16:9）效果最好
- 大小 1920px、< 500KB（用 inspace-case-deploy skill 一樣的壓縮規格）

## 想換照片要怎麼說

只要跟我說「首頁照片換掉」並把新照片放進這個資料夾，
我會自動：
1. 壓縮到 web 規格
2. 重新生成首頁 HTML（依照新張數）
3. push 上線

## 想加減張數

直接增加或刪除檔案。如果新增 `07.jpg`，跟我說「首頁加到 7 張」即可。
